# Đăng ký học phần UIT
## How to build
Yêu cầu cài đặt:
- golang
- make
Recommend: Linux, macOS

Chạy `make build`
## Cách sử dụng
Ví dụ: `./dkhp-uit -loop -1 -id 15520237 -pass "..;:?<>~%^&*()-_=+;[]" SE101.J11 SE221.J11.1 SE350.J11`

Trợ giúp: `./dkhp-uit -h`

Thuộc tính | Mô tả
--- | ---
*id* | Mã số sinh viên
*pass* | Mật khẩu tài khoản chứng thực
*loop* | Số vòng lặp (-1 để chạy vô tận)
